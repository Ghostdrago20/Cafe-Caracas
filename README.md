# Café Caracas App

**Café Caracas** es una aplicación móvil moderna y elegante desarrollada con **Ionic** y **Angular**, diseñada para ofrecer una experiencia digital premium a los clientes de nuestra cafetería.

## 🚀 Características Principales
*   **Menú Dinámico:** Un catálogo interactivo organizado por categorías (Cafés, Postres, Panadería) con imágenes atractivas.
*   **Diseño Premium:** Interfaz visualmente impactante con estética moderna, micro-animaciones y una experiencia de usuario fluida.
*   **Información del Negocio:** Sección dedicada a la historia, ubicación y contacto directo con la cafetería.
*   **Multiplataforma:** Construida para funcionar perfectamente en dispositivos móviles.

## 🛠️ Tecnologías
*   **Ionic Framework** - Desarrollo de interfaz móvil.
*   **Angular** - Lógica de la aplicación y componentes.
*   **Capacitor** - Despliegue nativo en dispositivos móviles.
